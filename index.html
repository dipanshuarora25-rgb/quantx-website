Core Features

QuantX Analytics gives you a complete toolkit to analyse any listed company across major markets using CPython and open data.

Multi-Exchange Support

Track companies on the London Stock Exchange (FTSE-style tickers), National Stock Exchange of India (NSE), and US markets (NYSE/Nasdaq) from one unified script.

Fundamental Ratios

Automatically compute net margin, ROE, current ratio, P/E, P/B and more using real financial statements, formatted for quick comparison.

Price History & Signals

Pull historical prices, compute moving averages, and generate clean CSVs you can plug into your own strategies, dashboards or reports.

How QuantX Analytics Works

1

Install the Toolkit

Download the project, create a virtual environment, and install dependencies with a single command using requirements.txt.

2

Set Your Markets & Tickers

Use the config.py file to define FTSE, NSE and NYSE watchlists, edit default periods and intervals, and customise your universe.

3

Run the Engine

Call the main script from your terminal to fetch price history, fundamentals and ratio snapshots for individual tickers or entire markets.

4

Analyse & Build on Top

Open the generated CSVs in Excel, Power BI or Jupyter, or extend the code to build dashboards, screeners, bots and full equity research reports.
